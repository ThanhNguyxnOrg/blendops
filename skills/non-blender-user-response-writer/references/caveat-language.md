# Caveat Language

## Purpose
Preserve caveat honesty while staying user-friendly.

## When to use
Whenever output certainty is partial.

## Pattern
- state uncertainty
- explain impact
- give next step

## Good example
"Runtime not executed, so export evidence is not available yet. Next: run official runtime eval."

## Bad example
"Everything should be fine."
