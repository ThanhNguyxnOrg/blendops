# Web Layout Negative Space

## Purpose
Ensure composition reserves space for web UI/copy overlays.

## When to use
Web/app hero output targets.

## Checklist
- copy-safe zone identified
- product silhouette not occluded
- responsive crop risk noted

## Good example
Explicit left/right negative space strategy with rationale.

## Bad example
Full-frame product leaves no text-safe zone.
