# Camera Rubric

## Purpose
Provide scoring lens for camera/composition planning quality.

## When to use
During quality review before handoff.

## Rubric dimensions
- focal clarity
- framing intent
- depth layering
- readability for target context

## Good example
Each dimension scored with pass/warn/fail rationale.

## Bad example
Single generic score with no category detail.
