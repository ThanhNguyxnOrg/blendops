# Hero Composition Patterns

## Purpose
Provide reusable hero composition patterns for product scenes.

## When to use
In composition planning stage.

## Patterns
- center-dominant product focus
- rule-of-thirds product offset
- diagonal dynamic hero composition

## Good example
Pattern chosen with rationale tied to user goal.

## Bad example
Pattern selected without relation to output use-case.
