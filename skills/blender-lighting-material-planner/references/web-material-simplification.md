# Web Material Simplification

## Purpose
Define simplification tiers for web-oriented handoff planning.

## When to use
When performance constraints are relevant.

## Tiers
- Tier 1: high-fidelity intent
- Tier 2: balanced portability
- Tier 3: conservative web-safe plan

## Good example
Tier chosen and justified in caveat section.

## Bad example
No simplification strategy despite performance constraints.
