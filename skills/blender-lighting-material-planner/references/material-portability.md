# Material Portability

## Purpose
Capture material assumptions that affect downstream portability.

## When to use
Before GLB/web handoff expectations are stated.

## Checklist
- PBR assumptions explicit
- unsupported complexity caveats noted
- simplification path documented

## Good example
Portability caveats listed with impact and next checks.

## Bad example
Assumes all material features transfer unchanged.
