# Reflective Floor Pattern

## Purpose
Plan reflective floor behavior without harming readability.

## When to use
Dark glossy floor scenarios.

## Pattern rules
- controlled reflectance assumptions
- highlight readability preservation
- fallback simplification if noisy

## Good example
Reflection supports mood while preserving subject edges.

## Bad example
Mirror-like floor overwhelms focal object.
