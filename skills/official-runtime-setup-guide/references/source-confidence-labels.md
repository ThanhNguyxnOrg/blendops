# Source Confidence Labels

## Purpose
Standardize confidence language across setup outputs.

## When to use
Any time runtime references are cited.

## Labels
- verified-read
- linked-only
- mixed
- unknown

## Pattern
Always pair label with a short reason.

## Good example
“linked-only: official URL confirmed, page content not fetched in this pass.”

## Bad example
“probably works” (no confidence label).
