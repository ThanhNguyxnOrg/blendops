# Clarification Patterns

## Purpose
Standardize high-value clarification questions.

## When to use
When prompt ambiguity affects output quality.

## Pattern checklist
- output destination
- fidelity priority
- model availability
- framing preference
- performance constraints

## Good example
Asks only questions that affect plan quality.

## Bad example
Asks broad irrelevant questions.
