# Scene Plan Schema

## Purpose
Define required structure for product hero planning output.

## When to use
For product-hero-scene-planner outputs.

## Required sections
- intent summary
- subject model
- composition plan
- camera plan
- lighting plan
- material plan
- environment plan
- render/export intent
- web handoff intent

## Good example
All sections present with assumptions and caveats.

## Bad example
Only composition and camera, missing handoff/validation.
