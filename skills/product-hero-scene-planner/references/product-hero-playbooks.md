# Product Hero Playbooks

## Purpose
Offer scenario playbooks beyond a single style.

## When to use
When user intent differs by product campaign type.

## Playbooks
- cyberpunk hero
- clean studio hero
- minimal e-commerce hero
- premium close-up detail hero

## Good example
Select playbook based on user audience/use-case.

## Bad example
Use one style for all requests.
