# Quality Rubric

## Purpose
Provide category rubric guidance for scene quality checks.

## When to use
During blender-scene-quality-checker scoring.

## Patterns
- evaluate by category before verdict
- tie each score to evidence/caveat state

## Good example
Each category has pass/warn/fail with rationale.

## Bad example
Single summary verdict without category detail.
