# GLB Handoff Checklist

## Purpose
Define minimum checks before GLB/web handoff claims.

## When to use
In handoff reporting stage.

## Checklist
- GLB status explicit
- preview status explicit
- assumptions and caveats explicit
- next verification actions explicit

## Good example
Includes Produced/Not Produced/Not Run status lines.

## Bad example
Claims handoff complete without artifact status.
