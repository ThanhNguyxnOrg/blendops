# Reference: Blender Quality Checklist

Status: Draft v0 reference

Use pass/warn/fail per category:

- subject clarity
- composition
- scale
- camera/framing
- lighting
- materials
- background/environment
- render readiness
- export readiness
- web readiness
- caveat reporting

## Usage note

When runtime is not executed:
- render/export/web categories should be marked Not Run or Warn with explicit caveats
- do not mark Ready without evidence
