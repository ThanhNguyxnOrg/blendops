# non-blender-user-response-writer (reference summary)

## Purpose
Plain-language final response for non-Blender users.

## When to use
- stakeholder-facing summary is needed,
- final output must be clear and non-technical.

## When not to use
- user is a Blender expert and prefers technical detail.

## Output/evidence contract
Provide plain-language summary with runtime/artifact status and caveats.

## Handoff notes
This is typically the final step; no further handoff unless user requests deeper detail.
