# blendops-help (reference summary)

## Purpose
Route users to the next safe BlendOps step when intent/status is unclear.

## When to use
- user asks what to do next,
- multiple skills/docs could apply,
- readiness/evidence/package status is unclear.

## When not to use
- a narrower skill clearly applies.

## Output/evidence contract
Return one recommended route + runtime/artifact status + caveats.

## Handoff notes
Hand off to planning, readiness, evidence, or response-writer summaries as appropriate.
