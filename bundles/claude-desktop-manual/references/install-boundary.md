This bundle provides skill/workflow guidance only.

- install Blender,
- configure Claude Desktop Connector,
- configure official Blender MCP bridge/add-on,
- configure third-party runtime bridges,
- run Blender or runtime eval,
- create/render/export/claim preview/render/GLB artifacts.

Claude Desktop import/copy is a manual user action. Do not assume AI UI automation.
