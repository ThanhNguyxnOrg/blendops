# blendops-help

Purpose: route users to the next safe BlendOps step when goal/status is unclear.

Use when users ask what to do next, which BlendOps path applies, or whether runtime/artifact/package status is safe to claim.

Do not use when a narrower planning, readiness, evidence, or handoff reference clearly applies.

Output contract: one recommended route, runtime status, artifact status, evidence used, caveats, and next safe action.
